# Package Entity / Model
Tempat menaruh semua kelas Model/Entity JPA yang merepresentasikan tabel di H2 Database.

## Yang perlu dibuat di sini:
- `Pendakian.java` (Data kelompok pendaki, waktu check-in/out, status)
- `MasterSampah.java` (Data jenis sampah global dan tarif dendanya)
- `DetailSampah.java` (Log bawaan sampah per kelompok kelompok pendaki)
- `User.java` / `Petugas.java` (Data login petugas pos)
